# 🎓 AI Accessible Evangadi Community Forum

An AI-powered, accessibility-first discussion platform built with React, Express.js, MySQL, and Google's Gemini AI.

This project extends the Evangadi Community Forum by integrating modern AI capabilities and accessibility tools to create an inclusive learning experience for all users.

---

## 🌟 Features

### 🤖 AI Features
- AI-powered semantic search
- Private PDF Knowledge Base (RAG)
- AI-generated answers using Gemini
- AI Presentation Studio
- AI-generated presentation slides

### ♿ Accessibility Features
- Voice command navigation
- Speech-to-text
- Text-to-speech (Screen Reader)
- Hover Reader
- Large Text mode
- High Contrast mode
- Keyboard navigation
- Presentation mode for accessible presentations

### 💬 Community Features
- User registration and login
- JWT Authentication
- Ask and answer questions
- Vote on answers
- Accept the best answer
- Personal dashboard
- User profile
- Knowledge Base management

## 🚀 Installation

### Clone the repository

```bash
git clone https://github.com/redseatransport/AI_Accessible_Evangadi_Forum.git
```

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

---

## 🔐 Environment Variables

Create a `.env` file inside the **backend** folder.

Example:

```env
PORT=3777

DB_HOST=localhost
DB_USER=your_database_user
DB_PASS=your_database_password
DB_NAME=your_database_name

JWT_SECRET=your_secret
JWT_EXPIRES_IN=1d

GEMINI_API_KEY=your_gemini_api_key
GEMINI_TEXT_MODEL=gemini-2.5-flash-lite
GEMINI_EMBEDDING_MODEL=gemini-embedding-001
```
